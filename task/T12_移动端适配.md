# T12 - 移动端适配 (Mobile Adaptation)

## 阶段
**Phase 7: 适配优化**

## 目标
优化移动端体验，为后续原生 App 打包做准备。

## 任务清单

### 12.1 响应式布局
- [x] Canvas 自适应屏幕尺寸
- [x] 保持游戏区域宽高比
- [x] 横屏/竖屏支持（通过 CSS object-fit: contain 适配）

### 12.2 触摸优化
- [x] 增大触摸目标区域（最小 44x44px）
- [x] 防止双击缩放
- [x] 防止滚动/拖拽干扰

### 12.3 移动端 UI 调整
- [ ] 选块按钮放在屏幕底部（保持桌面布局，依赖缩放）
- [ ] 按钮尺寸适合手指操作
- [ ] 信息面板简化

### 12.4 性能优化
- [ ] 降低动画复杂度选项
- [ ] 减少重绘区域
- [ ] 内存使用监控

### 12.5 PWA 支持（可选）
- [ ] manifest.json
- [ ] Service Worker（离线支持）
- [ ] 添加到主屏幕

### 12.6 原生打包准备
- [ ] 测试 Capacitor 集成
- [ ] 确保无浏览器特定 API 依赖
- [ ] 文档记录打包步骤

## 验收标准
- 在主流移动浏览器正常运行（Chrome, Safari）
- 触摸操作响应准确
- 无明显性能问题

## 预计工时
3-4 小时

## 依赖
- T10（UI 需要有移动端变体）
- T11（动画可能需要降级）
