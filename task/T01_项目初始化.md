# T01 - 项目初始化

## 阶段
**Phase 1: 基础设施**

## 目标
搭建项目基础结构，配置开发环境。

## 任务清单

### 1.1 创建 Vite + TypeScript 项目
- [ ] 使用 `npm create vite@latest` 初始化项目
- [ ] 选择 Vanilla + TypeScript 模板
- [ ] 安装依赖 `npm install`

### 1.2 项目目录结构
```
/src
  /core         # 核心游戏逻辑
  /ai           # AI 算法
  /render       # 渲染层
  /audio        # 音频系统
  /ui           # UI 组件
  /levels       # 关卡配置
  /utils        # 工具函数
  main.ts       # 入口文件
  style.css     # 全局样式
/public
  index.html
```

### 1.3 基础 HTML 结构
- [ ] 创建 Canvas 容器
- [ ] 设置响应式布局基础
- [ ] 配置 viewport meta 标签（移动端适配）

### 1.4 开发配置
- [ ] 配置 tsconfig.json（严格模式）
- [ ] 配置 vite.config.ts（路径别名）

## 验收标准
- `npm run dev` 可以启动开发服务器
- 页面显示一个空白 Canvas
- 控制台无报错

## 预计工时
1-2 小时

## 依赖
无
