# T11 - 动画效果 (Animations)

## 阶段
**Phase 6: UI/UX 打磨**

## 目标
添加流畅的动画效果，提升游戏体验。

## 任务清单

### 11.1 动画系统基础
- [ ] 创建 `src/render/Animator.ts`
- [ ] 实现缓动函数（easeIn, easeOut, easeInOut）
- [ ] 动画队列管理

### 11.2 方块动画
- [ ] **选择动画**：点击后方块飞向棋盘
- [ ] **下落动画**：方块平滑移动到目标位置
- [ ] **锁定动画**：方块落地时的"弹跳"或"闪光"
- [ ] **预览切换**：新的三选一方块滑入

### 11.3 消行动画
- [ ] 满行闪烁效果
- [ ] 行消除后上方方块下落动画
- [ ] 连消特效

### 11.4 UI 动画
- [ ] 界面切换淡入淡出
- [ ] 按钮悬停/点击效果
- [ ] 分数增加滚动数字

### 11.5 特效
- [ ] 粒子效果（消行时的碎片）
- [ ] 屏幕震动（Game Over）
- [ ] 胜利烟花效果

### 11.6 性能优化
- [ ] 使用 requestAnimationFrame
- [ ] 避免过多对象创建
- [ ] 可关闭动画选项（低性能设备）

## 验收标准
- 所有动画流畅无卡顿
- 动画可以被跳过或加速（避免等待过久）
- 60 FPS 运行

## 预计工时
4-5 小时

## 依赖
- T04（Renderer）
- T06（GameEngine 事件系统）
