# T11 - 动画效果 (Animations)

## 阶段
**Phase 6: UI/UX 打磨**

## 目标
添加流畅的动画效果，提升游戏体验。

## 任务清单

### 11.1 动画系统基础
- [x] 创建 `src/render/Animator.ts`
- [x] 实现缓动函数（easeIn, easeOut, easeInOut）
- [x] 动画队列管理

### 11.2 方块动画
- [x] **选择动画**：点击后方块飞向棋盘
- [x] **下落动画**：方块平滑移动到目标位置
- [x] **锁定动画**：方块落地时的"弹跳"或"闪光"
- [x] **预览切换**：新的三选一方块滑入

### 11.3 消行动画
- [x] 满行闪烁效果
- [x] 行消除后上方方块下落动画
- [x] 连消特效

### 11.4 UI 动画
- [x] 界面切换淡入淡出
- [x] 按钮悬停/点击效果
- [x] 分数增加滚动数字

### 11.5 特效
- [x] 粒子效果（消行时的碎片）
- [x] 屏幕震动（Game Over）
- [x] 胜利烟花效果

### 11.6 性能优化
- [x] 使用 requestAnimationFrame
- [x] 避免过多对象创建

## 验收标准
- 所有动画流畅无卡顿
- 动画可以被跳过或加速（避免等待过久）
- 60 FPS 运行
