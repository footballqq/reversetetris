## 阶段
**Phase 2: 核心逻辑**

## 目标
使用 Canvas 绘制游戏画面，包括网格、方块、UI 元素。

## 任务清单

### 4.1 Renderer 类基础
- [x] 创建 `src/render/Renderer.ts`

- [x] 实现自适应屏幕尺寸（保持宽高比）

### 4.2 绘制方法
- [x] `clear()`: 清空画布
- [x] `drawGrid(grid)`: 绘制网格背景和已锁定方块
- [x] `drawPiece(piece, x, y, rotation)`: 绘制活动方块
- [x] `drawGhostPiece(piece, x, y, rotation)`: 绘制幽灵块（落点预览）
- [x] `drawUI(state)`: 绘制分数、关卡信息等

### 4.3 选块 UI

### 4.4 视觉效果
