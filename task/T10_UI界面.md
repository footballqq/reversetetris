# T10 - UI 界面 (UI System)

## 阶段
**Phase 6: UI/UX 打磨**

## 目标
设计并实现游戏的完整 UI 界面，包括菜单、HUD、弹窗等。

## 任务清单

### 10.1 主菜单
- [x] 游戏标题 "Reverse Tetris"
- [x] 开始游戏按钮
- [x] 选择关卡按钮
- [x] 随机模式按钮
- [x] 设置按钮（音量）
- [x] 背景动画（可选）

### 10.2 关卡选择界面
- [x] 1-100 关卡网格
- [x] 已通关/未通关/锁定状态
- [x] 显示最高分/星级评价

### 10.3 游戏内 HUD
- [x] 当前关卡
- [x] 目标进度（AI 需消除 X 行，当前 Y 行）
- [x] 已放置方块数
- [x] 暂停按钮
- [x] 三选一方块预览区

### 10.4 暂停菜单
- [x] 继续游戏
- [x] 重新开始
- [x] 返回菜单
- [x] 设置

### 10.5 结算界面
- [x] 胜利界面（恭喜！AI 被你玩死了）
- [x] 失败界面（AI 成功存活）
- [x] 统计数据（用时、放置方块数等）
- [x] 下一关 / 重试 按钮

### 10.6 视觉风格
```
- 主色调：深蓝/紫色渐变背景
- 方块风格：霓虹灯效果，光晕
- 字体：像素风或现代无衬线
- 动画：按钮悬停缩放，界面淡入淡出
```

## 验收标准
- 所有界面可以正常切换
- 按钮交互响应正确
- 视觉风格统一美观
