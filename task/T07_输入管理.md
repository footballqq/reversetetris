# T07 - 输入管理 (InputManager)

## 阶段
**Phase 4: 交互层**

## 目标
处理用户输入，支持键盘和触摸操作。

## 任务清单

### 7.1 InputManager 类
- [ ] 创建 `src/ui/InputManager.ts`
- [ ] 绑定 Canvas 事件监听

### 7.2 键盘输入
- [ ] 数字键 `1, 2, 3` 选择对应方块
- [ ] `4` 键选择第四个方块（付费功能）
- [ ] `P` 键暂停游戏
- [ ] `R` 键重新开始
- [ ] `Escape` 键返回菜单

### 7.3 鼠标输入
- [ ] 点击方块预览区域选择
- [ ] 悬停高亮效果
- [ ] 点击按钮（开始、暂停等）

### 7.4 触摸输入（移动端）
- [ ] 触摸选择方块
- [ ] 大按钮设计，适合手指点击
- [ ] 防止误触和多点触控问题

### 7.5 输入映射
```typescript
interface InputEvent {
  type: 'selectPiece' | 'pause' | 'restart' | 'menu';
  data?: any;
}

// 回调注册
inputManager.on('selectPiece', (index: number) => {
  gameEngine.selectPiece(index);
});
```

## 验收标准
- 键盘输入响应正常
- 鼠标点击准确识别目标
- 移动端触摸体验流畅
